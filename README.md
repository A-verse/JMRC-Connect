# Jaipur Metro Connect


## Project Overview

Jaipur Metro Connect is a web application designed to provide information and utilities for Jaipur Metro commuters and tourists. It features journey planning, station details, timings, smart card info, tourist spots, and more.


## Getting Started

### Prerequisites
- Node.js & npm ([install guide](https://github.com/nvm-sh/nvm#installing-and-updating))

### Installation

Clone the repository and install dependencies:

```sh
git clone <YOUR_GIT_URL>
cd jaipur-metro-connect
npm install
```

### Running Locally

Start the development server:

```sh
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser to view the app.


## Features
- Journey planner
- Metro map
